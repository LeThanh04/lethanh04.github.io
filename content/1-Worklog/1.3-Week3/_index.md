### Week 3 Objectives:

* Understand the Amazon EC2 service and related components to deploy a virtual server on the AWS platform.
* Learn how to configure Security Groups and connect to an EC2 instance via SSH.
* Understand the Amazon CloudWatch service for resource monitoring and practice deploying a simple website on EC2.

### Week 3 Achievements:

* Gain an understanding of the Amazon EC2 service and the role of EC2 in virtual server deployment, as well as key components such as:
 * AMI
 * Instance Type
 * Key Pair

* Successfully create an EC2 instance via the AWS Management Console, configure appropriate settings, and launch the server.

* Understand the function of Security Groups.

* Successfully connect to the EC2 instance via SSH, gain initial familiarity with the Linux environment, and perform basic server administration tasks.

* Successfully install Nginx on the EC2 instance and deploy a sample website.
