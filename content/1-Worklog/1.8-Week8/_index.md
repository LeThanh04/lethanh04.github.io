### Week 8 Achievements:

* Gain an understanding of the Amazon API Gateway service.

* Understand the API workflow: sending requests to Lambda for processing and returning results to the application.

* Familiarize yourself with the AWS Management Console and learn how to locate, access, and use services via the web interface.

* Understand the API deployment process and the use of Stages to manage deployment versions.
