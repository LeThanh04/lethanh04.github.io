**On this page** you will find the work log documenting the activities completed throughout the 12-week First Cloud Journey Internship Program (April 17, 2026 – July 10, 2026). This internship covered learning AWS Cloud technologies, reviewing fundamental concepts, planning the project, and deploying AWS infrastructure for the AWS BILLO project.


A standard internship work log is typically maintained over a period of approximately three months. The weekly activities are summarized as follows:

**Week 1:** [Introduction to AWS and its core cloud services.](1.1-week1/)

**Week 2:** [Hands-on practice with AWS CLI, AWS Identity and Access Management (IAM), and basic AWS resource management.](1.2-week2/)

**Week 3:** [Hands-on deployment and management of Amazon EC2 instances.](1.3-week3/)

**Week 4:** [Introduction to Amazon RDS and Amazon S3.](1.4-week4/)

**Week 5:** [Learning Amazon DynamoDB and the Serverless architecture model on AWS.](1.5-week5/)

**Week 6:** [Further exploration of Amazon DynamoDB and the Serverless architecture model on AWS](1.6-week6/)

**Week 7:** [Knowledge review, comprehensive hands-on practice, and project deployment preparation.](1.7-week7/)

**Week 8:** [Requirements analysis and Serverless architecture design for the AWS BILLO project.](1.8-week8/)

**Week 9:** [System development using AWS Lambda, Amazon API Gateway, Amazon DynamoDB, and Amazon S3.](1.9-week9/)

**Week 10:** [Finalizing technical documentation, updating project documentation, and optimizing the AWS BILLO project.](1.10-week10/)

**Week 11:** [Integration testing, bug fixing, and finalizing the AWS BILLO system.](1.11-week11/)

**Week 12:** [Project review, completion of the internship report, and final project presentation.](1.12-week12/)
