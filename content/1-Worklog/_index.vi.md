**Trong trang này** bạn sẽ tìm thấy nhật ký công việc được ghi lại trong suốt 12 tuần thực tập chương trình First Cloud Journey (17/04/2026 – 10/07/2026). Quá trình này bao gồm học tập về AWS Cloud Journey, ôn tập kiến ​​thức, lập kế hoạch dự án và triển khai cơ sở hạ tầng AWS cho dự án AWS BILLO.


Thông thường và cũng là tiêu chuẩn, một worklog được thực hiện trong khoảng 3 tháng (trong suốt thời gian thực tập) với nội dung các tuần như sau:

**Tuần 1:** [Làm quen với AWS và các dịch vụ nền tảng của AWS.](1.1-week1/)

**Tuần 2:** [Thực hành AWS CLI, IAM và quản lý tài nguyên cơ bản.](1.2-week2/)

**Tuần 3:** [Thực hành triển khai và quản lý Amazon EC2.](1.3-week3/)

**Tuần 4:** [Làm quen với Amazon RDS và Amazon S3.](1.4-week4/)

**Tuần 5:** [Tìm hiểu Amazon DynamoDB và mô hình Serverless trên AWS.](1.5-week5/)

**Tuần 6:** [Tìm hiểu Amazon DynamoDB và mô hình Serverless trên AWS.](1.6-week6/)

**Tuần 7:** [Ôn tập, thực hành tổng hợp và chuẩn bị triển khai dự án.](1.7-week7/)

**Tuần 8:** [Phân tích yêu cầu và thiết kế kiến trúc Serverless cho dự án AWS BILLO.](1.8-week8/)

**Tuần 9:** [Phát triển hệ thống với AWS Lambda, Amazon API Gateway, Amazon DynamoDB và Amazon S3.](1.9-week9/)

**Tuần 10:** [Hoàn thiện tài liệu kỹ thuật, cập nhật Documentation và tối ưu dự án.](1.10-week10/)

**Tuần 11:** [Kiểm thử tích hợp, sửa lỗi và hoàn thiện hệ thống AWS BILLO.](1.11-week11/)

**Tuần 12:** [ổng kết dự án, hoàn thiện báo cáo và trình bày sản phẩm.](1.12-week12/)
